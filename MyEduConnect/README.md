# MyEduConnect

MyEduConnect is a PHP/MySQL web application developed for the CCS6324 Final Assignment.  
It demonstrates secure vs insecure coding practices in web development.

1. Clone the Repository
git clone https://github.com/Nurzahidah16/CCS6324-Final-Assignment.git
Place the cloned folder inside your XAMPP htdocs directory:
C:\xampp\htdocs\MyEduConnect

2. Start XAMPP
Open XAMPP Control Panel.
Start Apache and MySQL services.

3. Create Databases
Go to phpMyAdmin http://localhost/phpmyadmin.

db_insecure
db_secure
